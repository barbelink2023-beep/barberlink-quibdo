# BarberLink Quibdó

BarberLink es una aplicación web (MVP) orientada a dispositivos móviles y de escritorio, diseñada para conectar barberos y peluqueros jóvenes con clientes que demandan servicios de barbería de manera rápida, segura y organizada en el municipio de Quibdó.

## Funcionalidades principales
- Registro y visualización de perfiles de barberos
- Búsqueda de barberos por ubicación
- Visualización de perfil profesional
- Sistema de reservas simuladas
- Contacto directo con el barbero
- Sistema de reseñas y calificaciones
- Interfaz basada en prototipo UX/UI desarrollado en Figma

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript Vanilla
- Diseño UX/UI en Figma

## Estado del proyecto
MVP funcional en fase de validación.
